# Wordpress

Wordpress on yksi helpoimmista ja suosituimmista tavoista luoda nettisivut. Se on työkalu jonka avulla pystyt luomaan nettisivuja, ilman aikaisempaa tietämystä koodaamisesta. 

Tässä ohjeessa käymme läpi nettisivun luomisen Wordpressin avulla askel kerrallaan.
