# Nimipalvelimet

Webhotelliemme nimipalvelimet ovat:
```
ns.bittivirta.fi
ns.bittivirta.eu
ns.bittivirta.us
```

# Nimipalvelinmuutos 2024 kesäkuu

Uudet nimipalvelimet otetaan käyttöön kaikille webhotelleille maanantaina 15.7.2024. Vanhat nimipalvelimemme toimivat edelleen 1.1.2025 asti, mutta suosittelemme päivittämään nimipalvelimet mahdollisimman pian, sillä joidenkin verkkotunnuspäätteiden toiminta (mm. .fi-pääte) vaatii täydellisesti toimivat nimipalvelimet.

Jos verkkotunnuksesi on rekisteröity Bittivirralle, päivitämme nimipalvelimet puolestasi 15.7.2024. Jos verkkotunnuksesi on rekisteröity muualle, joudut itse päivittämään nimipalvelimet. Voit myös vaihtoehtoisesti siirtää verkkotunnuksesi Bittivirralle, jolloin hoidamme nimipalvelinmuutoksen puolestasi.
