# Saittikone

Saittikone on verkkosivujen rakennustyökalu, joka sisältyy webhotelliin. Se on suunniteltu auttamaan käyttäjiä luomaan moderneja, responsiivisia verkkosivuja ilman, että heillä tarvitsee olla laajaa ohjelmointi- tai web-design kokemusta.

## Ominaisuudet
Saittikone tarjoaa tehokkaita työkaluja ja ominaisuuksia, jotka mahdollistavat laadukkaiden verkkosivujen luomisen helposti:

- **Vetä ja pudota -editori**: Luo sivuja intuitiivisesti vetämällä elementtejä haluamaasi kohtaan sivulla.
- **Responsiiviset mallit**: Kaikki luomasi sivut toimivat saumattomasti eri laitteilla, kuten pöytäkoneilla, tableteilla ja älypuhelimilla.
- **Esikatselu eri laitteilla**: Voit tarkastella, miltä sivusi näyttää eri laitteilla suoraan editorista, jolloinne voit tehdä muokkauksia tarpeen mukaan.
- **SEO-työkalut**: Paranna sivustosi näkyvyyttä hakukoneissa optimoimalla sisältö SEO-ystävälliseksi.

## Kenelle Saittikone Sopii?
Saittikone sopii erinomaisesti erilaisille käyttäjille, jotka tarvitsevat tehokkaan mutta helppokäyttöisen työkalun verkkosivujen luomiseen:

- **Yrittäjät**: Nopea ja helppo tapa rakentaa yrityksen verkkosivusto.
- **Markkinointiasiantuntijat**: Tehokas väline kampanjasivujen luomiseen ja hallinnointiin.
- **Web-suunnittelijat**: Mahdollista asiakkaiden sivustojen suunnittelu ja ylläpito ilman koodausta, sekä tarjoa asiakkaille helppokäyttöinen työkalu sivustojen päivittämiseen.

## Miten aloitan Saittikoneen käyttö?
Aloittaminen Saittikoneen kanssa on suoraviivaista:

1. **Tilaa**: Tilaa mikä tahansa webhotellipaketti. Saittikone sisältyy kaikkiin webhotellipaketteihimme.
2. **Avaa Plesk**: Kirjaudu sisään webhotellisi hallintaan, Pleskiin ja valitse Saittikone "Aloita"-näkymästä.
3. **Valitse malli**: Valitse valmiiden mallien joukosta se, joka parhaiten sopii tarpeisiisi tai aloita tyhjällä mallilla.
4. **Mukauta Sivustoasi**: Käytä vetä ja pudota -editoria lisätäksesi ja muokataksesi sivustosi elementtejä.
5. **Julkaise**: Kun olet tyytyväinen sivustoon, julkaise se maailmalle suoraan Webhotelliisi.

Mikäli sinulla on kysyttävää Saittikoneesta tai tarvitset apua sen käytössä, ota yhteyttä asiakaspalveluumme. Autamme mielellämme!
