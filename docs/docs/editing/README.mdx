# Muokkaaminen

Jos käsilläsi sattuu olemaan enemmän aikaa, ja haluat osallistua dokumentaation kehittämiseen, voit muokata tai lisätä dokumentaatioon sisältöä. Tämä onnistuu GitHubin kautta tekemällä muutoksia dokumentaatioon ja lähettämällä ne tarkastettavaksi.

Kannattaa kuitenkin muistaa, että kyseessä on Bittivirran dokumentaatio, joten muutokset julkaistaan vasta, kun ne on hyväksytty Bittivirran henkilökunnan toimesta. Suuremmissa muutoksissa kannattaa siis kysyä ensin Bittivirran henkilökunnalta, onko muutoksille tarvetta.

Lisäksi, dokumentaatio on Bittivirran immateriaalioikeuksien alaista ja repositorion lisenssinä on omistusoikeuslisenssi (proprietary), joten muutokset siirtyvät Bittivirran omistukseen, kun ne on hyväksytty.

## Muokkaaminen GitHubin kautta

Dokumentaation muokkaaminen tapahtuu GitHubin kautta. GitHub on palvelu, joka tarjoaa Git-versionhallintaa web-käyttöliittymän kautta. Jotta muokkaus voidaan tehdä ja lähettää, sinun tarvitsee haarauttaa (fork) dokumentaatio omalle GitHub-tilillesi, tehdä muutokset ja lähettää ne tarkastettavaksi. Kun muutokset on hyväksytty, ne yhdistetään (merge) Bittivirran dokumentaatioon ja muutokset tulevat näkyviin. Kun muutokset on yhdistetty, voit poistaa haarasi (branch) omalta GitHub-tililtäsi.

Prosessi haarautuksesta käydään läpi [Haarauttaminen](forking) -artikkelissa.
