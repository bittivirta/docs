# Tekstin muotoilu

Tekstiä muotoillaan Markdownin ja Reactin yhdistelmällä, eli MDX-muodossa. MDX on Reactin laajennus, joka mahdollistaa Markdownin käytön React-komponenttien sisällä. MDX-tiedostot päättyvät `.mdx`-päätteeseen.

Markdown on yksinkertainen merkintäkieli, jolla voidaan muotoilla tekstiä. Markdownin avulla voidaan esimerkiksi luoda otsikoita, listoja ja linkkejä. Markdownin käyttö on helppoa, ja sen avulla voidaan luoda helposti selkeitä ja yhtenäisiä tekstejä.

React-komponentit ovat käytännössä muotoilun makroja, jotka palauttavat valmiita muotoiluja. Komponenttien käyttö on helppoa, ja niiden avulla voidaan luoda monimutkaisempia muotoiluja helposti.
