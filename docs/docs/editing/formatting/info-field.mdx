# Tietokenttä

Voit lisätä tietokenttiä, joissa voit näyttää esim. yhteystietoja ja muita tietoja.

## Esimerkki

```mdx
<InfoField title="Otsikko">Tieto</InfoField>
<InfoField title="Otsikko" icon={["fas", "info-circle"]}>Tieto</InfoField>
<InfoField icon={["fas", "info-circle"]}>Tieto</InfoField>
```
