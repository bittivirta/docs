# Pikanäppäimet

Voit ilmaista pikanäppäimiä `<Hotkey />` -komponentilla, esimerkiksi seuraavasti.

```jsx
> Tallenna painamalla <Hotkey>ctrl+s</Hotkey> tai macilla <Hotkey>cmd+s</Hotkey>
```
**Joka tulostuu:**
> Tallenna painamalla <Hotkey>ctrl+s</Hotkey> tai macilla <Hotkey>cmd+s</Hotkey>
