# Ongelmien ilmoittaminen

Ongelmien ilmoittaminen on suuri osa osallistumisen mahdollisuuksia. Ongelmien ilmoittaminen onnistuu GitHubin kautta, jossa voit ilmoittaa ongelmista [issues](https://github.com/bittivirta/docs/issues) -osiossa. Ongelmien ilmoittaminen onnistuu myös sähköpostitse osoitteeseen [tuki@bittivirta.fi](mailto:tuki@bittivirta.fi).

## Ongelmien ilmoittaminen GitHubin kautta

1. Mene [issues](https://github.com/bittivirta/docs/issues) -osioon
2. Klikkaa <kbd>New issue</kbd> -nappi
3. Kirjoita otsikko ja kuvaus ongelmasta.
4. Klikkaa <kbd>Submit new issue</kbd> -nappi

Kirjoita ongelman otsikko ja kuvaus mahdollisimman tarkasti. Ongelman kuvaus voi sisältää esimerkiksi seuraavia asioita:

- Kirjoitusvirheet
- Virheelliset tiedot
- Toimimattomat linkit

Kerro, millä sivulla ongelma on ja mitä ongelma koskee. Jos ongelma koskee esimerkiksi tiettyä sivua, kerro sivun osoite. Jos ongelma koskee koko sivustoa, kerro sekin.

## Ongelmien ilmoittaminen sähköpostitse

Ongelmien ilmoittaminen onnistuu myös sähköpostitse osoitteeseen [tuki@bittivirta.fi](mailto:tuki@bittivirta.fi). Kirjoita viestiin ongelman kuvaus mahdollisimman tarkasti. Ongelman kuvaus voi sisältää esimerkiksi seuraavia asioita:

- Kirjoitusvirheet
- Virheelliset tiedot
- Toimimattomat linkit
