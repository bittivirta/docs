# Omistajan vaihto

Verkkotunnuksen omistajan vaihto onnistuu Bittivirralla asiakaspalvelun kautta. Voit vaihtaa verkkotunnuksen omistajaa ottamalla yhteyttä asiakaspalveluumme ja pyytämällä omistajan vaihtoa. Pyrimme lähettämään avaimen arkisin viimeistään 24 tunnin kuluessa. Omistajan vaihto on fi-verkkotunnuksella maksuton palvelu. Verkkotunnuksen tulee olla vaihdon aikana Bittivirralla.
