# Verkkotunnus

Verkkotunnus on ainutlaatuinen nimi, joka identifioi Internetissä löytyvän verkkoresurssin, kuten verkkosivuston. Se toimii käytännössä internetsivuston osoitteena, jonka avulla käyttäjät voivat saavuttaa sivuston helposti ja nopeasti. Verkkotunnukset vastaavat numeerisia IP-osoitteita, joiden kanssa ihmiset eivät yleensä työskentele suoraan, sillä ne ovat vaikeita muistaa. Domaninimi toimii käytännöllisempänä ja ihmisläheisempänä järjestelmänä.

## Verkkotunnuksen rakenne

Verkkotunnukset koostuvat yleensä seuraavista osista:

1. **Alidomain**: Vapaasti valittava ja usein sivuston eri osioita kuvaava osa, kuten `www`, tai vaikkapa `blogi`.
2. **Toisen tason domain (SLD, Second Level Domain)**: Yleisnimi, joka yleensä kuvaa yrityksen tai organisaation nimeä, esim. `google`.
4. **Ylätason domain (TLD, Top-Level Domain)**: Domain-nimen viimeinen osa, kuten `.com`, `.org`, tai maakohtainen päätteitä kuten `.fi` Suomen osalta.

### Esimerkki:

```
www.kirvesluoma.fi
aliverkkotunnus.verkkotunnus.pääte
```

## Verkkotunnuksen tarkoitus

Verkkotunnuksen pääasiallinen tarkoitus on tarjota helppo ja muistettava keino löytää ja viitata verkkoresursseihin. Sen avulla voidaan:

- **Edustaa organisaatiota verkossa**: Verkkotunnus voi viestiä yrityksen nimeä ja alaa, mikä helpottaa brändin tunnistettavuutta.
- **Parantaa käytettävyyttä**: Ihmiset voivat helpommin muistaa ja kirjoittaa selaimen osoiteriville sanoja sisältävän osoitteen kuin pelkistä numeroista koostuvan IP-osoitteen.
- **Suorittaa tehokasta markkinointia**: Yksilöllinen ja osuva domain-nimi voi tehostaa markkinointia ja auttaa erottumaan kilpailijoista.

## Miten verkkotunnus toimii teknisesti?

Internetin nimipalvelimet (DNS) toimivat verkkotunnusten välisten yhteyksien yhteydessä, kääntäen domain-nimet vastaaviksi IP-osoitteiksi, jotta selaimet voivat ladata halutut verkkosivut:

1. Käyttäjä kirjoittaa verkkotunnuksen selaimen osoitekenttään.
2. Selain tai tietokone kysyy DNS-palvelimelta, mikä on verkkotunnuksen IP-osoite.
3. DNS-palvelin vastaa IP-osoitteella, ja selain voi yhdistää kyseiseen IP-osoitteeseen palvelimelle, jossa sivusto sijaitsee.
4. Sivusto latautuu käyttäjän selaimelle.

## Verkkotunnusten hankinta

Verkkotunnusten rekisteröinnin voi tehdä monet yritykset, jotka toimivat rekisteröijinä. Hinta, säännöt ja ehdot vaihtelevat yleensä TLD:n ja palveluntarjoajan mukaan. Verkkotunnuksen hankkiminen on yleensä ensimmäinen askel verkkosivuston perustamisessa.

Voit tilata verkkotunnuksen Bittivirralta sivuiltamme osoitteessa https://bittivirta.fi/ ja hallita sitä kätevästi Portaalissamme.

## Yhteenveto

Verkkotunnus on kriittinen osa internetin navigointia. Se ei ainoastaan helpota sivustojen löytämistä ja saavutettavuutta vaan on myös keskeinen osa yritysten brändäystä ja internet-läsnäoloa.
