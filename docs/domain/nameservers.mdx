# Nimipalvelimet

Verkkotunnus tarvitsee nimipalvelimet toimiakseen internetissä. Nimipalvelimet ovat palvelimia, jotka kääntävät verkkotunnuksen IP-osoitteeksi, jotta selaimet voivat ladata halutut verkkosivut. Nimipalvelimet ovat välttämättömiä, jotta verkkotunnus toimii.

Voit tilata Bittivirralta Webhotellin maksutta käyttöösi, kun rekisteröit verkkotunnuksen meiltä. Voit hallita nimipalvelimia kätevästi sivuiltamme. Webhotelli sisältää nimipalvelimet ja pienen tilan verkkosivustolle.

## Nimipalvelimien asettaminen

Nimipalvelimet asetetaan tilauksen yhteydessä. Jos käytät Bittivirran Webhotellia, nimipalvelimet ovat valmiiksi asetettuina. Bittivirran nimipalvelimet ovat `ns1.bittivirta.fi`, `ns2.bittivirta.fi` ja `ns3.bittivirta.fi`.

## Nimipalvelinten muokkaaminen

Voit muokata nimipalvelimia Portaalissa. Kirjaudu sisään, valitse minun verkkotunnukset ja valitse verkkotunnus, jonka nimipalvelimia haluat muokata. Voit lisätä, poistaa ja muokata nimipalvelimia tarpeen mukaan.
