# Mikä on Java-versioni?

Java versiolla tarkoitetaan Minecraft serverisi Java versiota, löydät tiedot helposti Gamepanelista:

<Stepper>
    <Step>
    kirjaudu Gamepaneliin osoitteessa [https://gamepanel.fi](https://gamepanel.fi)
    <Image img={require("./assets/java-version/gamepanel.png")} />
    </Step>
    <Step>
    Gamepanel aukeaa 'Dashboard' näkymään josta voit valita klikkaamalla haluamasi serverin
    <Image img={require("./assets/java-version/dashboard.png")} />
    </Step>
    <Step>
    Valittuasi haluamasi serverin, navigoi ylhäältä 'Startup' osioon (1). Näet Java versiosi numeron 'Docker image' osiossa (2)
    <Image img={require("./assets/java-version/startup.png")} />
    </Step>
    <Step>
    Voit vaihtaa Java versiota klikkaamalla 'Docker image':ssa olevaa dropdown menua, ja valita haluamasi Java version
    <Image
    style={{maxWidth:"400px"}}
    img={require("./assets/java-version/java-version.png")} />
    </Step>
</Stepper>
