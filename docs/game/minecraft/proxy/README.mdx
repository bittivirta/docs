# Proxy-palvelin

Proxy-palvelimet tarjoavat sujuvan ratkaisun useiden pelipalvelimien yhdistämiseen. Kuvittele tilanne, jossa "yhdellä" keskitetyllä palvelimella hallinnoit kolmea erillistä Minecraft-palvelinta: Palvelin A, Palvelin B ja Palvelin C. Jokainen näistä palvelimista toimii itsenäisenä yksikkönä, mahdollistaen erilaiset pelimuodot ja ainutlaatuiset kokemukset.

Ilman proxy-palvelinta, kunkin palvelimen välillä siirtyminen edellyttäisi palvelimelta poistumisen ja toiselle liittymisen eri osoitteella. Proxy-ohjelmiston avulla voit kuitenkin luoda saumattoman yhteyden A, B ja C palvelinten välille, mikä tekee palvelimelta toiselle siirtymisestä vaivatonta, komennolla tai lisäosan avulla esimerkiksi portaaleilla tai klikkaamalla tavaraa.

Tämän kätevän yhteyden aikaansaamiseksi tarvitset proxy-palvelimen, joka onneksi ei ole suuri investointi. Riittää, että varaat sille vähintään 512 RAM-muistia. **Bittivirta tarjoaa sinulle perustason proxy-palvelimen veloituksetta**, kunhan sinulla on käytössäsi vähintään kaksi Minecraft-palvelinta. Tämä perustason palvelin vastaa useimpien tarpeisiin, mutta mikäli kaipaat lisää suorituskykyä, voit helposti päivittää palvelimesi kapasiteettia.
