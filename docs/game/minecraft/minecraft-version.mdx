# Mikä on Minecraft-versioni?

Minecraftista on olemassa kaksi versiota: Java Edition ja Bedrock.
Minecraft Bedrock koostuu yhdistelmästä eri Minecraft versioita, kuten esim. kaikki konsoleiden Minecraftit sekä Minecraft Windows 10:lle. Java Edition- version ostaneille on automaattisesti asentunut Bedrock- versio, Java- version asennuksen yhteydessä. Jos pelaat Mac:illä ilman emulaattoria, versiosi on Java Edition.

Jos haluat pelata Minecraftia moninpelinä, tulee kaikkien pelaajien pelata samaa versiota, eli esim. Minecraft Bedrock pelaaja ei voi pelata Minecraft Java Edition pelaajan kanssa, koska pelien versiot eivät ole yhteensopivia.
