# Oma .jar-tiedosto

Joskus saatat haluta käyttää omaa .jar-tiedostoa esim. modeja pelatessa tai ihan
muuten vain haluat käyttää toista palvelinohelmistoa Minecraft-palvelimesi
ylläpitoon.

Tämä onnistuu yksinkertaisimmillaan siten, että poista vanha
`server.jar`-tiedosto palvelimen File Manager-kohdassa ja lataa tilaa uusi.
Käynnistettävän .jar-tiedoston tulee olla aina saman niminen, eli `server.jar`.
