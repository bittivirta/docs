# Minecraft

Minecraft on Mojang AB:n kehittämä ja julkaisema hiekkalaatikkomainen videopeli. Minecraftin PC-versio julkaistiin 18. marraskuuta 2011. Minecraftia on myyty yli 200 miljoonaa kappaletta ja se on yksi maailman myydyimmistä videopeleistä. Minecraftin kehitystä johti Markus "Notch" Persson vuoteen 2011 asti, jonka jälkeen Jens "Jeb" Bergensten otti ohjat käsiinsä. Vuonna 2014 Microsoft osti Mojangin ja Minecraftin oikeudet 2,5 miljardilla dollarilla.

Videopeliä voidaan pelata joko yksin tai kavereiden kanssa moninpelinä palvelimella. Bittivirta tarjoaa helposti hallittavia Minecraft-palvelimia, johon voit asentaa haluamasi lisäosat ja muokata pelin asetuksia.

Bittivirran tarjoaman palvelimen hankkiminen on helppoa. Tilaa palvelin ja palvelin asentuu automaattisesti. Sinulle jää vain pelaaminen ja halutessasi voit muokata pelin asetuksia. Suosittelemme kuitenkin ottamaan ainakin palvelimen Whitelistan (pääsylistan) käyttöön, jotta vain haluamasi pelaajat pääsevät palvelimelle ja varmuuskopioimaan palvelimen tiedostot säännöllisesti, jotta et menetä pelaajien rakentamia rakennelmia.
