# Pluginit

Minecraftin pluginit ovat lisäosia, joita voidaan käyttää pelin palvelimilla. Ne tarjoavat uusia ominaisuuksia ja toimintoja, jotka eivät ole mukana pelin tavallisessa versiossa.

Toisin kuin modit, pluginit asennetaan palvelimelle, eli ne eivät muokkaa itse pelin tiedostoja. Pluginit keskittyvät enemmän olevassa olevien ominaisuuksien hyödyntämiseen ja vuorovaikutukseen pelaajien kesken, kun taas modit muokkaavat peliä laajemmin ja voivat lisätä täysin uusia ominaisuuksia ja elementtejä peliin.

Bittivirran Minecraft-palvelimet käyttävät oletuksena Purpur ohjelmistoa.
