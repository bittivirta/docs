# Tekstin värit ja muotoilut

Minecraftissa voit monessa paikassa käyttää värikoodeja. Värikoodeja voi käyttää
esim. MOTD-teksteissä ja Essentials-lisäosan avulla chatissa ja kylteissä.

MOTD-tekstiä lisäteessäsi server.properties-tiedostossa, voit käyttää §-koodin
värikoodeja tekstin muotoiluun.

Saat `&` -merkin useimmilla näppäimistöillä painamalla <Hotkey>shift+6</Hotkey>.
`§` -merkin taas saat useimmilla näppäimistöillä painamalla itse merkin näppäintä,
joka sijaitsee useimmiten <Hotkey>esc</Hotkey>-näppäimen alla.
