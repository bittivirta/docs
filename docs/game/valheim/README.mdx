# Valheim

Valheim on Ruotsalaisen Iron Gate AB:n kehittämä ja Coffee Stainin julkaisema selviytymispeli. Peli julkaistiin Steamin Early Access -ohjelmassa 2. helmikuuta 2021. Peli on myynyt yli 7 miljoonaa kappaletta ja on yksi Steamin pelatuimia pelejä.

Bittivirta tarjoaa Balheim-palvelimia, jonka avulla voit pelata ystäviesi kanssa ja muokata palvelimen asetukset haluamallasi tavalla. Voit päivittää palvelimen suorituskykyä tarpeen mukaan.
