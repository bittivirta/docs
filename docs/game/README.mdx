# Pelipalvelimet

Pelipalvelin on palvelu, joka tarjoaa pelaajille mahdollisuuden pelata yhdessä.
Tarjoamme monelle eri pelille palvelimia, joita sinun on helppo hallita tarjoamassamme
hallintapaneelissa.

Pelipalvelimien hallinta tapahtuu Gamepanel-hallintapaneelissa, johon pääset kirjautumaan
[https://gamepanel.fi](https://gamepanel.fi) sivustolla.

:::info
Huomaathan, että Gamepanelin salasana on eri kuin asiakastilisi salasana. Sähköposti on kuitenkin sama.
:::
