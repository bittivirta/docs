# Käynnistysparametrit

Tässä artikkelissa ohjeistetaan palvelimen parametreissa. Pääset muokkaamaan
asetuksia kun navigoit:

Palvelimesi » (valitse haluamasi palvelin) » Startup

Kaikki alla listatut parametrit ovat asiakkaan itse muokattavissa.

Alla listatuista parametreista RCON vaatii pientä säätämistä, ohjeet siihen
täältä: [tulossa]

Saat itse asettaa oman verkkosivuston palvelimelle tai määrittää itse
pelaajamäärän!

| Parametri               | Kuvaus                                                                                                                     | Vakioarvo                   |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| Palvelimen nimi         | Palvelimen nimi julkisella palvelinlistalla                                                                                | A Rust Server               |
| OxideMod                | Haluatko ottaa OxideMod -lisäosan käyttöön?                                                                                | FALSE                       |
| Maailman tyyppi         | Halutessasi voit itse omalla vastuulla tutustua muihin maailmatyyppeihin. Vakioarvoa suositellaan.                         | Procedural Map              |
| Palvelimen kuvaus       | Kuvaus palvelimesta palvelimen nimen alla. Yleensä sisältää tietoa palvelimesta tai sääntöjä. Kirjoita vaihtaaksesi riviä! | Powered by Bittivirta.cloud |
| Verkkosivusto           | Linkki palvelimen verkkosivustoon. Ohjaa sivulle kun pelissä klikkaa "Visit Website" kohtaa.                               | Linkki sivustolle           |
| Maailman koko           | Määrittele generoimasi maailman koko. Mahdolliset arvot 1000 - 6000.                                                       | 3000                        |
| Maailman Seed           | Seed kartan generaatiota varten.                                                                                           | 1                           |
| Palvelimen kuvake       | Palvelimen otsakekuvake.                                                                                                   | Linkki kuvaan               |
| RCON Salasana           | RCON yhteyttä varten käytössä oleva salasana.                                                                              | Piilotettu!                 |
