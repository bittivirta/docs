# Rust

Rust on Facepunch Studiosin kehittämä moninpeli, jossa pelaajat yrittävät selviytyä alkeellisessa maailmassa. Pelaajat voivat kerätä resursseja, rakentaa suojia, metsästää ja taistella muita pelaajia vastaan.

Bittivirta tarjoaa Rust-palvelimia, jonka avulla voit pelata ystäviesi kanssa ja muokata palvelimen asetukset haluamallasi tavalla. Voit päivittää palvelimen suorituskykyä tarpeen mukaan.
