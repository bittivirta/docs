# Asiakastili

Asiakastili on asiakkuutesi Bittivirralla. Asiakastiliin liitetään palvelut, laskut ja tukipyynnöt. Asiakastilin käyttö edellyttää käyttäjätilin luomista. Käyttäjätili luodaan aina samalla, kun luodaan asiakastili. Asiakastilin yhteydessä luotu käyttäjätili on asiakastilin pääkäyttäjätili. Pääkäyttäjätili voi luoda ja hallinnoida muita käyttäjätilejä.
