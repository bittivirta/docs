# Asiakastilin vaihtaminen

Yhdellä käyttäjätilillä voi olla useampi asiakastili. Voit vaihtaa asiakastiliä
seuraavasti:

<Stepper title="Asiakastilin vaihtaminen">
  <Step>
    Kirjaudu käyttäjätilillesi osoitteessa
    [portal.bittivirta.fi/login](https://portal.bittivirta.fi/login)
  </Step>
  <Step>
    Valitse "Vaihda asiakastili"
    <Image
      style={{ maxWidth: 600 }}
      img={require("./assets/asiakastili.png")}
    />
  </Step>
  <Step>
    Valitse asiakastili, jolle haluat kirjautua klikkaamalla "Vaihda
    asiakastili"
    <Image img={require("./assets/asiakastilit.png")} />
  </Step>
</Stepper>
