# Käyttäjätili

Käyttäjätili on tili, jolla hallinnoidaan asiakastiliä ja asiakastilin palveluita, laskuja ja tukipyyntöjä. Käyttäjätiliin kirjaudutaan käyttäjätunnuksella ja salasanalla. Käyttäjätiliin liittyy yksi tai useampi asiakastili.

Asiakastilillä voi olla useampi käyttäjätili ja yksi käyttäjätili voi olla liitetty useampaan asiakastiliin.
