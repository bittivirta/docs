# Käyttäjätilin poistaminen

Mikäli haluat, että poistamme käyttäjätilisi, ota yhteyttä asiakaspalveluumme. Mikäli omistat asiakkuuden, tiliäsi ei voida poistaa ilman, että poistamme myös asiakkuuden.

Tiedot poistuvat lopullisesti viimeistään vuoden kuluessa, kun yli vuoden vanhat tiedot poistetaan varmuuskopioinnin piiristä. Aktiivisessa järjestelmässä olevat tiedot poistuvat välittömästi.

Jos olet asiakastilin omistaja, noudata ohjeita kohdassa [Asiakastilin poistaminen](../client/delete-account).


## Käyttäjätilin poistaminen

Lähetä sähköposti osoitteeseen [gdpr@bittivirta.fi](mailto:gdpr@bittivirta.fi) ja pyydä käyttäjätilisi poistamista.

Kirjoita viestiin, että ymmärrät ja hyväksyt, että poistamalla käyttäjäsi, tietosi poistetaan lopullisesti eikä niitä voida palauttaa. Mikäli tämä jää mainitsematta, kysymme vielä varmistuksen ennen kuin poistamme käyttäjäsi.
