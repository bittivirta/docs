# Kirjautuminen

Sähköpostitiliin kirjautuminen onnistuu joko kolmannen osapuolen sähköpostiohjelmalla tai webmailin kautta. Webmail on selainpohjainen sähköpostiohjelma, joka mahdollistaa sähköpostien lukemisen ja lähettämisen suoraan selaimessa.

## Webmailiin kirjautuminen

Webmailiin pääset osoitteessa [https://webmail.bittivirta.fi](https://webmail.bittivirta.fi). Kirjaudu sisään sähköpostiosoitteellasi ja salasanallasi. Jos olet unohtanut salasanasi, voit nollata sen ohjeiden mukaan: [Salasanan nollaaminen](./reset-password).

## Sähköpostiohjelman käyttäminen

Voit käyttää sähköpostitiliäsi myös kolmannen osapuolen sähköpostiohjelmalla, kuten Mozilla Thunderbird, Microsoft Outlook, Apple Mail tai Gmail. Näissä ohjelmissa sinun tulee määrittää sähköpostitili manuaalisesti tai automaattisesti riippuen ohjelman tarjoamista ominaisuuksista.

### Sähköpostimääritykset

Sähköpostin määritykset ovat seuraavat:

| Asetus            | Arvo                      |
| ----------------- | ------------------------- |
| Palvelin (isäntä) | `plesk.bittivirta.fi`     |
| Portti, IMAP      | 993                       |
| Portti, SMTP      | 465                       |
| Salauksen tyyppi  | SSL/TLS                   |
| Käyttäjätunnus    | sähköpostiosoitteesi      |
| Salasana          | Sähköpostitilisi salasana |

Jos haluat käyttää POP3-protokollaa, käytä porttia 995 IMAPin sijaan. Muista, että POP3 lataa sähköpostit paikallisesti laitteellesi ja poistaa ne palvelimelta, ellei toisin ole määritetty. IMAP puolestaan synkronoi sähköpostit palvelimen kanssa, joten voit käyttää sähköpostia useilla laitteilla ilman, että viestit katoavat. Lisäksi IMAP tukee sähköpostin kansioita ja merkkauksia, kun taas POP3 ei.

POP3-protokollan portti on 995 ja SSL/TLS-salaus on suositeltavaa käyttää turvallisuuden takaamiseksi.
