# Nimipalvelimet

DNS-nimipalvelimet (Domain Name System) ovat keskeisiä osia internetin infrastruktuurissa, kun puhutaan webhotelleista. Ne toimivat ikään kuin internetin puhelinluetteloina, jotka auttavat muuttamaan verkkosivustojen nimet (esim. www.kirvesluoma.fi) niitä vastaaviksi IP-osoitteiksi (esim. 95.217.166.192), jotka ovat tarpeen, jotta selaimet voivat löytää ja ladata kyseiset sivustot.

Webhotellin yhteydessä DNS-nimipalvelimet ovat erityisen tärkeitä, koska ne varmistavat, että verkkosivusi on löydettävissä internetissä. Kun hankit webhotellipalvelun Bittivirralta, saat käytöösi kolme DNS-nimipalvelintamme. Nämä nimipalvelimet tulee lisätä verkkotunnuksen rekisteröijän palveluun, jotta domainisi osoittaa oikeaan webhotelliin ja sen sisältöön internetissä.

DNS-nimipalvelimet webhotellissa hoitavat myös muita tehtäviä, kuten sähköpostien reitityksen oikeisiin palvelimiin ja alidomainien hallinnan, jotka ovat usein tarpeen monimutkaisemmissa verkkopalveluissa.

Webhotelliemme nimipalvelimet ovat:
```
ns1.bittivirta.fi
ns2.bittivirta.fi
ns3.bittivirta.fi
ns4.bittivirta.fi
```
