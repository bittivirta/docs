# Tietokannat

Tietokannat ovat järjestelmiä, joilla voidaan tallentaa, hallita ja järjestellä suuria määriä tietoa. Voit ajatella tietokantaa digitaalisena arkistona, jossa kaikki tiedot ovat järjestyksessä ja helposti saatavilla. Yleisin tietokantaohjelmisto on MySQL, mutta sen pohjalta kehitetty MariaDB on yritysystävällisempi sekä sisältää joitain uudempia ominaisuuksia.

Saatat tarvita tietokantaa erilaisten verkkosivusovellusten käyttämiseen, kuten esimerkiksi WordPress. Täältä löydät ohjeet MariaDB:n, eli täysin MySQL:ää vastaavan palvelinohjelmiston asennukseen.
