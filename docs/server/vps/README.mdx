# Virtuaalipalvelimet

Kun et halua tilata itsellesi kokonaista palvelinta, mutta tarvitset silti palvelintilaa, voit tilata virtuaalipalvelimen. Virtuaalipalvelin on suuremmalla palvelimella sijaitseva virtualisoitu palvelin. Virtuaalipalvelimella on omat resurssit, kuten muisti, prosessori ja levytila. Virtuaalipalvelimella on myös oma käyttöjärjestelmä, joka on eristetty muista virtuaalipalvelimista. Virtuaalipalvelimella on myös oma IP-osoite, joka on vain sinun käytössä.

Bittivirta tarjoaa virtuaalipalvelimia eri tyyppisillä prosessoreilla. Voit valita virtuaalipalvelimen, jossa on joko AMD:n, Amperen tai Intelin prosessori. Prosessori on mahdollista valita joko jaettuna tai dedikoituna. Jaettu prosessori tarkoittaa, että prosessorin resurssit jaetaan useamman virtuaalipalvelimen kesken. Dedikoitu prosessori tarkoittaa, että prosessorin resurssit ovat vain sinun käytössäsi.
